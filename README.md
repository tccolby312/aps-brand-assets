# APS Brand Assets

Public hosting for Assurance Power Systems brand images (logos), used in email
signatures and other materials. Served via GitHub Pages.

- `email/` — email-optimized logos (300px, transparent PNG)
- `logos/` — full-resolution original logos (SVG + PNG, black & white)

Colors: red `#D12130` · gold `#FCD605` · ink `#231F20`.
